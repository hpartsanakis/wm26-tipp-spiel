# 🏆 WM26 Tipp-Spiel

Ein modernes Web-App-Tippspiel für die FIFA World Cup 2026.

Mit dieser Anwendung können Gruppenspiele und KO-Spiele getippt, Ergebnisse eingetragen und Punkte automatisch berechnet werden.

---

# ✨ Features

## Gruppenphase

- 12 Gruppen (A–L)
- 48 Mannschaften
- 72 Gruppenspiele
- Gruppentabellen
- Automatische Tabellenberechnung
- Tore, Tordifferenz und Punkte

## Tippspiel

- Tipp pro Spiel speichern
- Tipp löschen
- Ergebnisse speichern
- Ergebnisse löschen
- Speicherung im Browser (LocalStorage)

## Punktewertung

| Regel | Punkte |
|---------|---------|
| Exaktes Ergebnis | 3 |
| Richtige Tordifferenz | 2 |
| Richtiger Sieger / Unentschieden | 1 |
| Falsch | 0 |

## KO-Phase

- Runde der letzten 32
- Achtelfinale
- Viertelfinale
- Halbfinale
- Finale

## Dashboard

- Anzahl Spiele
- Anzahl Tipps
- Offene Tipps
- Gesamtpunkte
